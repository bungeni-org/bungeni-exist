Terminology
Upload

Loading configuration from bungeni (on file-system) onto configuration manager (on eXist-db).

These imports will be listed on the upload sub-menu of upload/commit top-menu.

Each import provides options for activating or deleting it. See screenshot below
The import coloured green is the active imported for editing on the configuration manager.

Commit

Writing back changes made on configuration manage (on eXist-db) onto bungeni (on file-system).

The same location on the file-system where it was imported from.
