Bungeni ConfigEditor 0.1 XRX application

ConfigEditor is an application use to manage Bungeni configuration files and saves them back to the file system. It allows basic CRUDS (create, read, update, delete and search) actions.

This application uses the XRX (XForms Rest XQuery) web application architecture. Essentially this means that all functionality was built using XML technologies from front to back. To learn more about XRX please refer to the XRX Wikibook
